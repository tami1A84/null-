# null² v0.1.0 リリースノート

**リリース日**: 2025年12月6日

## 🎉 初回リリース

nostr-loginとAmber対応のLINE風Nostr Webクライアント

## ✨ 新機能

### 認証機能
- ✅ **nostr-login統合** - 複数の認証方法をサポート
- ✅ **Nostr Connect (NIP-46)** - nsec.app等のbunkerで安全に認証
- ✅ **ブラウザ拡張 (NIP-07)** - nos2x, Alby等に対応
- ✅ **ローカル署名** - 秘密鍵を直接入力（ブラウザに保存）

### コア機能
- ✅ **投稿機能** - yabu.meリレーに直接投稿
- ✅ **タイムライン** - yabu.meリレーの最新投稿を表示
- ✅ **プロフィール編集** - yabu.meリレーに送信
- ✅ **いいね** - kind 7イベント送信
- ✅ **リポスト** - kind 6イベント送信

### UI/UX
- ✅ **LINE風デザイン** - 馴染みのあるインターフェース
- ✅ **レスポンシブ** - モバイル・デスクトップ対応
- ✅ **4つのタブ** - ホーム・トーク・タイムライン・おさいふ
- ✅ **LINE Seedフォント** - 読みやすい日本語フォント

## 🔧 技術的な変更

### アーキテクチャの簡略化
- **設定ファイル不要** - nostr-loginがすべて処理
- **バックエンド簡略化** - 静的ファイル配信のみ
- **依存関係削除** - Go標準ライブラリのみ
- **直接リレー通信** - フロントエンドからWebSocketで直接接続

### セキュリティ向上
- **秘密鍵の非サーバー管理** - サーバーは秘密鍵を一切扱わない
- **複数の認証方法** - ユーザーが好みの方法を選択可能
- **bunkerサポート** - 最も安全な認証方法をサポート

## 📝 使い方

### インストール

```bash
git clone https://github.com/tami1A84/null-web.git
cd null-web
make run
```

### ブラウザでアクセス

```
http://localhost:8080
```

### ログイン

画面右上の「ログイン」ボタンをクリックして、お好きな方法でログイン。

# null² クイックスタート

## 🚀 3ステップで開始

### 1. サーバー起動

```bash
cd null-web
go run cmd/main.go
```

### 2. ブラウザで開く

```
http://localhost:8080
```

### 3. ログイン

右上の「ログイン」ボタンをクリック！

## 🔐 ログイン方法

### 方法1: ブラウザ拡張機能

1. nos2x、Alby、Flamingo等をインストール
2. 「ログイン」→「Sign in with extension」
3. 完了！

### 方法2: Nostr Connect

1. 「ログイン」→「Sign in with Nostr Connect」
2. nsec.app等でQRコードをスキャン
3. 承認して完了！

### 方法3: Amber（Android）

1. Amberアプリをインストール
2. 「ログイン」→Amberを選択
3. Amberアプリで承認
4. 完了！

### 方法4: 読み取り専用

1. 「ログイン」→「Read-only」
2. npubを入力
3. タイムライン閲覧のみ可能

## 📝 投稿する

1. タイムラインタブを開く
2. テキストボックスに入力
3. 「投稿」ボタンをクリック
4. yabu.meリレーに送信される！

## 👤 プロフィール編集

1. ホームタブを開く
2. 「編集」ボタンをクリック
3. 名前、自己紹介、アイコンURLを入力
4. 「保存」ボタンをクリック
5. yabu.meリレーに送信される！

## 🔧 トラブルシューティング

### ログインできない

- ブラウザ拡張機能がインストールされているか確認
- Nostr Connectの場合、QRコードが表示されているか確認
- Amberアプリが起動しているか確認

### 投稿できない

- ログインしているか確認
- yabu.meリレーに接続できているか確認（コンソールログを確認）
- ブラウザのコンソールでエラーを確認

### タイムラインが表示されない

- yabu.meリレーに接続できているか確認
- ブラウザのコンソールでエラーを確認
- ページをリロードしてみる

## 💡 ヒント

- ログイン後は自動的にプロフィールとタイムラインが読み込まれます
- 投稿は即座にyabu.meリレーに送信されます
- プロフィール編集も即座にyabu.meリレーに送信されます
- 複数のアカウントを切り替えることができます（nostr-login機能）

## 📱 モバイルで使う

1. モバイルブラウザで開く
2. Amber（Android）でログイン
3. そのまま使える！
